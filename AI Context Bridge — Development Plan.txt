AI Context Bridge — Development Plan
Vision
Build a browser extension that allows users to move AI conversations between:
* Claude Chat → New Claude Chat
* Claude Account A → Claude Account B
* Claude → ChatGPT
* ChatGPT → Claude
* ChatGPT/Claude → Cursor/Windsurf
without manually rebuilding project context.
All project contexts live:
User Browser
   ↓
IndexedDB
Users can:
* Export
* Delete
* Backup
their own data.
MVP Scope
Only solve:
Use Case 1
Claude Chat
↓
New Claude Chat
________________

Data Model
interface ProjectContext {
   id: string;


   title: string;


   sourcePlatform: string;


   createdAt: number;


   updatedAt: number;


   messages: Message[];
}


interface Message {
   role: "user" | "assistant";


   content: string;


   timestamp?: number;
}

Module 1: Claude Extractor
Responsibility
Read Claude conversations.
________________


Steps
1. User clicks: Save Context
2. Content script:
* Reads current conversation
* Extracts all messages
3. Converts to: ProjectContext
4. Stores locally
________________


Important
Extractor must be isolated.
Example: extractors/claudeExtractor.ts
Because Claude's UI will eventually change.
________________

Module 6: Context Loader
Used when continuing conversations.
________________


Flow:
Select Project
      ↓
Open New Claude Chat
      ↓
Load Context
________________


Loader:
1. Retrieves project
2. Retrieves chunks
3. Injects chunk(s) into chat input
________________


Important:
Never auto-send.
Only populate input.
User presses Enter.
________________
